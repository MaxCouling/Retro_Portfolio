## Overview

SurveyHustle is a platform enabling ethical, privacy-focused data sharing through anonymised surveys.

### Key Features
*   **Privacy-First:** Integrates Differential Privacy to protect user data.
*   **Full Stack:** Built with Python (Flask) and PostgreSQL.
*   **Self-Hosted:** Deployed on a Raspberry Pi protected by Cloudflare Tunnels.

### Tech Stack
*   Python (Flask)
*   PostgreSQL
*   Docker
*   Cloudflare Tunnels
