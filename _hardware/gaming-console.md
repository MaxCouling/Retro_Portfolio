## Overview

This project involved building a gaming console from scratch, moving beyond just software to understanding the hardware layer.

### The Build
*   **PCB Design:** Designed a custom PCB to handle button inputs and power management.
*   **Fabrication:**
    *   **Buttons:** 3D printed for custom tactile feel.
    *   **Chassis:** Laser-cut acrylic case for a retro-industrial look.
*   **Assembly:** Hand-soldered all components.
